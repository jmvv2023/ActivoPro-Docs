# 1.1 Clientes / Usuarios

Para gestionar la ficha de un cliente:

**Menú → Contactos → Clientes/Usuarios**

!!! note "ALTA RÁPIDA"
    Para dar de **ALTA** un nuevo cliente, haga clic en **Nuevo**.  
    El sistema propondrá un identificador automático.

## Pestañas de la ficha

- **General**: datos básicos (nombre, NIF/CIF)
- **Condiciones de Cobro**
- **Cuentas**
- **Información Adicional**
- **FACe**
- **Facturas**
